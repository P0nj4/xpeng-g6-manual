• Cuando el vehículo emite una advertencia, el conductor

La advertencia de colisión frontal incluye la advertencia

Si el riesgo aumenta aún más, y el conductor sigue
sin frenar o la fuerza de frenado es demasiado pequeña,
el vehículo frenará de forma activa para reducir las colisiones
y las lesiones.
